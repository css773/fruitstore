# React-Native+Mobx开发一个简单的水果商城App

> 此项目是基于react-native + mobx + react-navigation 来开发的，主要用mobx来模拟实现 **浏览商品 => 加入购物车=> 结账 => 清空购物车 => 还原商品初始状态** 流程

相关依赖：
NPM 版本6.10.3
Node 版本12.10.0
需要先在本地安装node


# 项目运行

```

cd fruitstore

yarn

android: react-native run-android
ios: react-native run-ios
```

# 实现功能

* 商品预览
* 加入购物车
* 模拟支付
* 清空购物车，还原商品初始状态
* 页面的跳转和传值
